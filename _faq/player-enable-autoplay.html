--- category: player type: tutorial title: "Enable Autoplay" ---

The autoplay features has been added to NewPipe recently. We wanted to keep the UI as clean as possible. Therefore you can only enable an disable this feature via the settings:

  1. Open the settings.
  2. Open the Content category.
  3. Toggle the Auto-queue next stream option to append a related stream automatically when the stream starts. This has no effect on repleating play queues.
